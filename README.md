## Project Overview

The purpose of this portfolio is not to present myself as a finished professional, but to show the direction I am developing toward. The site focuses on how I want to combine design, technology, AI tools, and creative strategy into a believable professional identity.

The portfolio is structured like a horizontal presentation-style website. It highlights my professional direction, project concepts, developing skills, and AI workflow.

## Professional Direction

My intended direction is centered around AI product thinking, digital design, and creative systems. I am interested in how AI can be used as a serious workflow tool for research, planning, writing, design refinement, and project development.

## Tools Used

## AI Workflow

AI was used to help research professional directions, refine the writing, organize the site structure, improve the design concept, and think through how the portfolio should communicate a realistic future-facing identity.

## Project Goal

The goal of this project is to create a portfolio that feels intentional, believable, and professionally grounded while still being aspirational.
